# Project: RobDelivery - Robotic Delivery Management System

## Description:
Full-stack system for automating last-mile delivery using autonomous robots and drones. Includes backend API, web client, IoT drone module, and admin analytics.

## Role:
Full-Stack Developer

## Technologies:
ASP.NET Core 8.0, C#, Entity Framework Core, SQLite, REST API, JWT, Swagger, Docker, React, TypeScript, Vite, SignalR, Python (Raspberry Pi IoT), Google Maps API

## Key Features:
- Clean Architecture (4 layers: Entities, Infrastructure, Application, API)
- JWT authentication with role-based access (User, Admin, IoT)
- Google OAuth integration
- Order management with automatic pricing calculation
- Robot fleet management (ground robots, aerial drones)
- Real-time robot telemetry via SignalR
- IoT module for drone control (Python/Raspberry Pi)
- Payment integration (PayPal, Google Pay, Stripe)
- Admin dashboard with statistics and analytics
- Database backup and delivery history export
- Google Maps integration for route visualization
- Docker containerization
